# Sources

This skill was built from the following official MongoDB resources:

## Primary Sources

1. **MongoDB Presentation Style Guide** (PDF)
   - Official brand guidelines for MongoDB presentations
   - Covers: visual approach, typography, colors, logos, architecture diagrams, icons (technical, content, events, security, cloud, numbers/nodes, industry, miscellaneous), spot illustrations, photography, accessibility
   - Published by MongoDB brand team

2. **MongoDB Presentation Template** (PDF, 161 pages)
   - Complete catalog of approved slide layouts
   - Covers: welcome/title, agenda, chapter breaks, big bold statements, text slides, code slides, quote slides, icon-based, illustration-based, photography-based, image-based, webpage/screenshot, case study, timeline, charts and graphs (tables, bar charts, line charts, donut charts, process diagrams, bubble charts, matrices), partner/language logos, closing slides
   - Each layout available in dark (Slate) and light (White) variants
   - Published by MongoDB brand team

## Supplementary Sources

3. **MongoDB LeafyGreen Design System** — https://www.mongodb.design/
   - Web/UI design system (used for general brand color reference only)
   - The presentation style guide takes precedence for all presentation-specific rules
   - Color palette reference: https://www.mongodb.design/foundation/palette/
